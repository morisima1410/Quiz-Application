# 📱 Quiz Application

An Android Quiz Application developed using **Java** and **Android Studio**. The application provides an interactive quiz experience with multiple-choice questions, score calculation, and a user-friendly interface.

## 🚀 Features

* 📝 Multiple Choice Questions
* ⏱️ Quiz Timer
* 📊 Instant Score Calculation
* 📱 Clean & Responsive UI
* 🔄 Restart Quiz Option
* ⚡ Fast Performance

## 🛠️ Tech Stack

* Java
* Android Studio
* XML
* Firebase *(if used)*
